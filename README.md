# AI SEO Blog & Content Cluster (Task 3)

This project demonstrates how prompt engineering can be used to generate SEO-optimized blog content using a local SEO strategy.

## Concept
This project targets students in Hyderabad looking for acne treatment solutions by combining SEO structure and prompt engineering.

## SEO Keywords
- acne treatment for students in Hyderabad
- best facewash for acne Hyderabad
- affordable acne solutions for students
- skincare routine for students India
- how to reduce acne fast for students

## Features
- SEO blog outline (structured headings)
- Full blog content
- Meta title and description
- FAQs section
- Content cluster (internal linking)

## Tools Used
- ChatGPT
- Prompt Engineering Techniques

## Outcome
Created a complete SEO blog system with local targeting and structured content for better engagement and search visibility.
